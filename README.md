# 🧮 Modern Calculator

A sleek, responsive, and fully functional calculator built with **HTML5, CSS3, and Vanilla JavaScript**. Designed with a modern dark UI, keyboard support, smooth animations, and robust calculation logic.

![Calculator Preview](./assets/calculator-preview.png)

---

## ✨ Features

### Core Functionality

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* 📊 Percentage calculations
* 🧹 Clear All (AC)
* ⌫ Delete Last Character
* 🔢 Decimal support

### User Experience

* 🎨 Modern dark-themed interface
* 📱 Fully responsive design
* ⌨️ Full keyboard support
* ⚡ Smooth button animations
* 🔍 Dynamic display resizing
* 🚫 Division-by-zero error handling
* 📈 Automatic number formatting

### Advanced Features

* Chained calculations
* Smart percentage calculations
* Floating-point precision handling
* Expression history display
* Error recovery system
* Mobile-friendly touch interactions
